# Metamorfose Landing Page

Landing page moderna para Metamorfose - Marketing é Transformação.

## 🚀 Deploy no Vercel

### Pré-requisitos
- GitHub account
- Vercel account

### Passos para Deploy:

1. **Crie um repositório no GitHub:**
   - Vá para github.com e crie um novo repositório
   - Clone-o localmente

2. **Adicione os arquivos:**
   ```bash
   # Copie todos os arquivos do projeto para o repositório
   git add .
   git commit -m "Initial commit: Metamorfose landing page"
   git push origin main
   ```

3. **Faça o deploy no Vercel:**
   - Vá para vercel.com
   - Clique em "New Project"
   - Selecione seu repositório do GitHub
   - Vercel detectará automaticamente que é um site estático
   - Clique em "Deploy"

4. **Pronto!** 🎉
   - Sua landing page estará disponível em `https://seu-projeto.vercel.app`

## 📁 Estrutura do Projeto

```
.
├── index.html          # Landing page principal
├── package.json        # Configurações do projeto
├── vercel.json         # Configurações do Vercel
├── .gitignore         # Arquivos a ignorar

```

## 💻 Desenvolvimento Local

Para rodar localmente:

```bash
npm run dev
```

A página abrirá em `http://localhost:8000`

## 📝 Customizações

Para editar a landing page, abra o `index.html` em seu editor de código favorito.
